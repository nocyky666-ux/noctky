# NOCTKY UNIVERSE

Ultra-cinematic dimensional portfolio experience for Noctky.

## Tech Stack

- **Next.js 14** — App Router
- **Three.js + React Three Fiber** — WebGL universe
- **GSAP + Framer Motion** — Cinematic animations
- **Tailwind CSS** — Utility styling
- **TypeScript** — Type safety

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

1. Push to GitHub
2. Import project to [vercel.com](https://vercel.com)
3. Deploy — no env variables needed
